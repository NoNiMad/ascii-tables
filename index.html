Text To ASCII Tables

Cette petite application vous permet de convertir du texte quelque peu formatté en un tableau fait en ASCII Art. Plusieurs paramètres sont actuellement disponibles dans le code (skins notamment) et viendront s'ajouter à l'interface prochainement.
Merci de signaler les bugs à @NoNiMad.

Pour créer une nouvelle colonne, utilisez le symbole "|".
Pour créer une nouvelle ligne, utilisez le symbole "-".
